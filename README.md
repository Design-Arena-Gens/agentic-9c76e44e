## اثغ | Echo Playground

تجربة تفاعلية مصممة بواجهة واحدة تستكشف إمكانات الحروف المجرّدة. تتحول كلمة «اثغ» عبر طبقات من التكرار، اللون، والنبض، ويمكن التحكم في كثافة ظهورها ودرجة تموّجها عبر منزلقات حيّة.

### تشغيل محلي

```bash
npm install
npm run dev
```

ثم افتح المتصفح على [http://localhost:3000](http://localhost:3000) لمشاهدة التجربة.

### بناء للإنتاج

```bash
npm run build
npm run start
```

### ملاحظات

- تم بناء الواجهة باستخدام Next.js (App Router) وخط Geist الافتراضي.
- العنصر الرئيسي `EchoVisualizer` يعرض شبكة حروف ديناميكية مع مؤثرات تموّج وضوء.
- جاهز للنشر على Vercel مباشرةً.
